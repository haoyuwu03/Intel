Folder Design:
-> \YOLOv3\
--> train.py: used for uploading the dataset and training
--> data.py: used for changing the dataset into usable format
--> yolov3.py: used to hold the yolov3 architecture
--> utils.py: used to hold the functions required for display and other utils
--> test.py: used to test 
--> demo.py: used for demo, will be similar to application implementation
--> \YOLOv3\checkpoints\: holds the checkpoints
--> \YOLOv3\dataset\: holds the dataset
